## Task

將對話中的英文或簡體內容重寫成**繁體中文**。

### Arguments

```
$ARGUMENTS
```

### 目標選擇

| 參數                      | 翻譯目標             |
| ------------------------- | -------------------- |
| 無參數                    | 上一則回覆           |
| 有描述（如 "上面的報告"） | 對話中符合描述的內容 |

### 要求

| 項目 | 說明                                             |
| ---- | ------------------------------------------------ |
| 語言 | 繁體中文（台灣用語）                             |
| 術語 | 技術術語、程式碼、檔案名保持原文                 |
| 格式 | 保留原有的 markdown 格式（表格、代碼塊、列表等） |
| 內容 | 完整重寫，不省略任何部分                         |

### 轉換規則

- 簡體 → 繁體（詞彙轉換，非單純字體）
- 英文說明 → 繁體中文
- 保持專業術語一致性

### 執行

直接輸出重寫後的完整內容，無需額外說明。
