# Review Rubric

## Severity

| Level | 定義                           | 例子           |
| ----- | ------------------------------ | -------------- |
| P0    | 安全漏洞、資料破壞、核心不可用 | SQLi, 權限繞過 |
| P1    | 正確性風險、效能退化、測試缺口 | 競態, N+1      |
| P2    | 設計瑕疵、可維護性問題         | 過深嵌套       |
| Nit   | 風格、命名                     | 變數名         |

## Merge Gate

| Gate       | 條件                          |
| ---------- | ----------------------------- |
| ✅ Ready   | 無 P0，P1 已修或有 mitigation |
| ⛔ Blocked | 有 P0，或 checks failing      |
